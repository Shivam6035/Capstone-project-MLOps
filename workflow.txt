-------------------------Setting up project structure---------------------------

1. Create repo, clone it in local
2. Create a virtual environment named 'cap' - conda create -n cap python=3.10
3. Activate the virtual environment - conda activate cap
4. pip install cookiecutter ( for making project structure it will automatically create all files etc.)
5. cookiecutter -c v1 https://github.com/drivendata/cookiecutter-data-science
6. Rename src.models -> src.model
7. git add - commit - push
